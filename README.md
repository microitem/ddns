# DDNS Server pre DS223

Vlastný DDNS (Dynamic DNS) server špeciálne navrhnutý pre Synology DS223 NAS.

## Funkcie
- PowerDNS server
- Webové API
- MySQL backend
- Docker kontajnery
- Automatické zálohovanie
- Monitoring systém
- SSL/TLS podpora
- Bezpečnostné nastavenia
